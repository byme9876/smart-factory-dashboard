# Smart Factory Monitoring Dashboard

A full-stack industrial monitoring system developed using Flask, SQLite, HTML/CSS, and JavaScript. The application simulates smart factory operations by monitoring machine status, temperature, and alerts in real time through interactive dashboards and REST APIs.

## Features
- Real-time machine monitoring
- Temperature and status visualization
- REST API integration
- User authentication system
- Interactive charts using Chart.js
- SQLite database integration
- Responsive dashboard UI
- GitHub version control and deployment support
